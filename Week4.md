# 실증적 AI 개발 프로젝트 4주차 회의록

**회의일시:** 2025.04.02  
**회의장소:** S03-210  
**참석자:** 정호진, 강다영, 조동건

## 회의 주요 내용

- 선행 연구 발표
- Transform Mode 예측의 필요성 및 기대 효과 분석
- 입력 feature 후보 검토

## 결정사항

- 주변 블록 IPM, residual energy를 주요 입력으로 선정
- 출력 class는 MTS index(0~4)로 정의

## 차주 계획

- Transform Mode label 추출 코드 개발
- MLP 구조 설계 착수
